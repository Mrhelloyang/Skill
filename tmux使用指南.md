# Tmux使用指南

## 1.Tmux会话的操作

> 1.创建会话：tmux new -s [用户名]
>
> 2.退出会话："Ctrl +b"   +	d
>
> 3.查看有哪些会话：tmux ls
>
> 4.进入已有的会话：tmux  a -t  [用户名]
>
> ==5.在会话内切换会话："Ctrl +b"   +	S==
>
> 6.重命名会话："Ctrl +b"   +	$
>
> 7.删除已有的会话：tmux kill-session -t [用户名]

## 2. Tmux窗格操作

> ==1.添加窗格："Ctrl +b"   +	c==
>
> ==2.切换窗格："Ctrl +b"   +	窗格序号==
>
> ==3.删除窗格："Ctrl +b"   +	x==
>
> 4.左右分屏："Ctrl +b"   +	%
>
> 5.上下分屏："Ctrl +b"   +	"
>
> 6.跳转到10的窗格："Ctrl +b"   +	’
